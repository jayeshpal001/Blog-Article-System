# Blog-Article-System

A full-stack Blog and Artical plateform built with the MERN stack. It features role-based access control, JWT
authentication using HTTP-only cookies, and complete CRUD operations for posts and comments. 

## Live Links 
-**Frontend (Vercel):**[https://blog-article-system.vercel.app]
-**Backend (Render):**[https://blog-article-system.onrender.com]

## Tech Stack 
-**Frontend:** React, Tailwind CSS
-**Backend:** Node.js, Express.js
-**DataBase:** Mongodb
-**Security:** JWT(Json Web Token), Brypt, cookie-parser

## Features
-**Role-Based Authentication:**  -Readers can read published posts and comment.
-Writers can create, edit, and delete their own posts. 
-Admins can manage all posts and comments.
-**Security:** JWTs are securely stored in HTTP-only coolies. 
**Search & Pagination:** API supports fetching published posts with query parameters
**Comments System:** Authenticated users can leave comments; authors/admins can delete them.
